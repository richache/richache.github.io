正则表达式的概念

  • 正则表达式是一种字符串搜索模式。
  • 由一个字符串来描述，用于匹配搜索目标中符合特定句法规则的子字符串。
  • 格式：/pottern/modifiers
  • /pottern/ 就是一个字符串（搜索模式）
  • /modifiers 就是一个修饰符（进一步的匹配条件）

正则的使用

  • 常用search()检索特定子字符串并返回它的起始位置；使用replace()寻找并替换某个指定的子字符串
  • 也可以使用RegExp来调用预设的方法。在 JavaScript 中，RegExp 对象是一个预定义了属性和方法的正则表达式对象。

一些实例

使用search()寻找子字符串World在“Hello World”中的位置，返回起始位置

var searchworld = "Hello World";

var str = searchworld.search(/World/);


当直接使用字符串作为参数时，字符串参数会转换成正则表达式,如下

var searchworld = "Hello World";

var str = searchworld.search("World");


使用replace()在“Hello World”中寻找子字符串World并用JS替换它，无返回值

var replaceworld = "Hello World";

var str = replaceworld.replace("World","JS");

document.write(RegExp("e").test("Hello World"));
