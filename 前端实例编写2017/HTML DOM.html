HTML DOM的概念

DOM全称Document Object Model,也就是文档对象模型。当网页被加载时，浏览器就会创建页面的DOM，通过DOM可以访问页面文档的所有元素。

在HTML页面中通过可编程的DOM可以实现：（动态页面功能）

  • 改变页面所有HTML元素；
  • 改变页面所有HTML元素属性；
  • 改变页面所有CSS样式；
  • 对页面所有事件做出反应。

DOM如何获取节点

为了准确描述和定位我们需要的元素，我们称目标元素为一个节点。一个节点也就是一个元素（标签）。

元素（标签）里面可以嵌套其他的元素，因而节点里面也可以有子节点。

为节点添加id或class（类名），以便DOM访问。

抽象来说，DOM的工作原理就是创建一个DOM对象树，通过getElementByXXX()的DOM方法访问指定节点然后加以改变。

我的理解：DOM通过建模将页面（Document对象）分割成一个个元素（Element对象，父节点），每个元素里面也可以分割出更多内容（属性或文本，子节点）。 给每个节点赋予名字用于访问，声明变量将访问获取的节点储存起来，然后就可以方便地使用预设方法对节点进行编程 ，实现增删改查操作。

有三种方法可以找出（访问）目标节点：

  • document.getElementById()方法，通过寻找Id定位节点（常用）
  • document.getElementsByClassName()方法，通过寻找类名定位节点（注意：Internet Explorer 8 及更早 IE 版本不支持该方法。）
  • document.getElementsByTagName()方法，通过寻找标签名定位节点（应配合前两种使用，使定位的节点更为精准）

p元素节点的id是id1

p元素节点的class是class1

实例：
		var a=document.getElementById("id1");
		a.innerHTML="该节点通过id获取。";
		
		var b=document.getElementsByClassName("intro");
		b.innerHTML="该节点通过class获取。";
