概念与特点

JSON 英文全称 JavaScript Object Notation，是用于存储和传输数据的格式。通常用于服务端向网页传递数据 。

  • JSON 是一种轻量级的数据交换格式。
  • JSON 是独立的语言
  • JSON 易于理解。

格式

{"JSON数组对象名":[{"键":"值","键":"值","键":"值"......},{"键":"值","键":"值","键":"值"......},......]}

  • 中括号内为一个JSON数组
  • 数组内每一个大括号包含的范围都是一个JSON对象（数组内全是匿名对象，所以指定时要使用数组脚标）
  • JSON对象内包含着一个或者多个键/值对，键/值对就是JSON的数据格式（类似于JS中的对象属性）
  • 最外面的大括号把整个JSON数组包含了起来，这是一个完整的JSON数组对象（可转换为JS对象）

转换成JS对象

使用JS内置函数JSON.parse()可以将JSON字符串转换为一个JS对象

显示指定数据

		源代码：
		var text='{"employees":['+'{"firstname":"Steve","lastname":"Jobs"},'+'{"firstname":"Bill","lastname":"Gates"}]}';
		var obj=JSON.parse(text);
		function demo(){
			document.getElementById("demo").innerHTML=obj.employees[0].firstname+" "+obj.employees[0].lastname;
		}
