变量

JavaScript 使用关键字 “var” 来定义变量，使用等号来为变量赋值。

【字面量和变量的区别：】变量可以通过变量名访问。在指令式语言中，变量通常是可变的。字面量是一个恒定的值。

【小结:】变量是一个名称，字面量是一个值。

例如：
function sum(a,b){return a+b;}
var num1=5,num2=10;
document.write(sum(num1,num2));

其他需要注意的地方

【分号】JavaScript中的语句必须要以分号;作结尾。

【命名】JavaScript中变量的命名遵从驼峰原则。

【关键词】JavaScript 语句通常以关键词为开头。 var 关键词告诉浏览器创建一个新的变量。

【标识符】JavaScript 保留了一些关键字，这些关键字在当前的语言版本中并没有使用，但在以后 JavaScript 扩展中会用到。

JavaScript 标识符必须以字母、下划线（_）或美元符（$）开始。

后续的字符可以是字母、数字、下划线或美元符（数字是不允许作为首字符出现的，以便 JavaScript 可以轻易区分开标识符和数字）。

【注释】JavaScript 不会执行注释。

我们可以添加注释来对 JavaScript 进行解释，或者提高代码的可读性。

单行注释以 // 开头，多行注释以 /* 开始，以 */ 结尾。
