# Markdown
本文讲述Markdown的定义及使用方法，并有样式演示
* 轻量
* 是一种基本的**标记语言**（不是编程语言，不能表达逻辑，只能表达信息）
* 内容是带格式的纯文本
  
# 语法
1. N级标题，示例如下（编辑时前面加#，或者在次行加=或-）：
# #一级标题 或者次行写===
## ##二级标题 或者次行写---
### ###三级标题
#### ####四级标题
##### #####五级标题
###### ######六级标题 （最多六级

2. 段落 （段落不需换行。隔一个空行，区分两个段落）
3. 链接 （中括号内为表面内容，小括号为具体链接）
  [linktext](url)
4. 图片 ![linktext](url)
5. 加粗 **双星号*或双_**，斜体 *单星号*
6. cite引用  >
7. 列表  
    * 无序：*+- 
    * 有序：num. 
    * 层级：列表里面缩进4个空格
8. 代码  3个反引号```(引号后可声明代码语言)一段代码
   
  ```js
  function add(a,b){
    return a+b*2
  }
  ```

9.  表格  | ---: | :--- | :---: |

| 学号  | 姓名 |  分数 |
| :---:| :---:| :---:|
| 1 | 张三 | 90 |
| 2 | 李四 | 95 |
| 3 | 王五 | 98 |

    * checklist :  -[ x ]
